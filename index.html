Hardware Design

UWB Anchor System

System Architecture

The anchor network provides centimeter-level indoor positioning using Ultra-Wideband (UWB) technology. Anchors are strategically placed throughout the facility to create overlapping coverage zones, enabling continuous 3D position tracking with <10cm accuracy.

Why UWB Over WiFi/BLE Positioning?

Accuracy: UWB provides 5-10cm accuracy vs 1-5m for WiFi/BLE triangulation. Latency: <50ms update rate vs 1-3s for signal strength methods. Penetration: UWB signals are less affected by walls and obstacles. Multi-path Resistance: Time-of-flight measurement is immune to reflections that plague RSSI-based systems.

Anchor Placement Optimization Algorithm

Problem Definition

Determining the optimal number and placement of UWB anchors is a complex optimization problem. The goal is to minimize the number of anchors while ensuring complete coverage and acceptable positioning accuracy (GDOP < 3) throughout the facility.

Mathematical Formulation

Coverage Constraint

‚àÄ point p ‚àà Coverage_Area:
   |{anchor_i : distance(p, anchor_i) ‚â§ R_max}| ‚â• 4

where:
R_max = maximum effective range (25m with walls)
Minimum 4 anchors required for 3D positioning

Geometric Dilution of Precision (GDOP)

GDOP(p) = sqrt(trace((A^T √ó A)^(-1)))

where A is the observation matrix:
A[i] = [(x_i - x_p)/r_i, (y_i - y_p)/r_i, (z_i - z_p)/r_i, 1]

Constraint: GDOP(p) < 3 for all critical points p

Lower GDOP = Better positioning accuracy
GDOP < 2: Excellent (5-8 cm accuracy)
GDOP 2-3: Good (8-12 cm accuracy)
GDOP > 3: Poor (>15 cm accuracy)

Optimization Algorithm

Hybrid Approach: Genetic Algorithm + Local Refinement

1. INITIALIZATION
- Import floor plan (CAD/PDF/image)
- User marks: walls, doorways, critical zones, no-mount areas
- Define constraints: ceiling height, power outlet locations
- Generate initial population of anchor configurations

2. FITNESS FUNCTION
fitness(config) = w1√ócoverage + w2√ó(1/avg_GDOP) + w3√ó(1/num_anchors) + w4√óinstallation_ease

where:
- coverage: % of area with ‚â•4 anchor visibility
- avg_GDOP: average GDOP across sampled points
- num_anchors: total anchor count (minimize cost)
- installation_ease: proximity to power, mounting feasibility
- weights: w1=0.4, w2=0.35, w3=0.15, w4=0.10

3. GENETIC OPERATORS
- Selection: Tournament selection (k=5)
- Crossover: Uniform crossover with repair (ensure feasibility)
- Mutation: Gaussian perturbation (œÉ=0.5m) + anchor add/remove
- Population: 100 individuals, 200 generations

4. LOCAL REFINEMENT
- Apply gradient descent on GDOP minimization
- Enforce minimum spacing constraints (>3m between anchors)
- Snap to mounting grid (align with ceiling tiles/walls)

5. OUTPUT
- Optimal anchor positions (x, y, z coordinates)
- Coverage heatmap with GDOP visualization
- Installation plan: mounting hardware, cable routing
- Estimated cost: anchors + installation labor

Real-World Constraints

Physical Constraints

  • Power outlet availability (25m max cable run)
  • Mounting surface requirements (concrete, drywall, suspended ceiling)
  • Aesthetics & visibility (minimize visual impact)
  • Interference zones (elevators, metal structures)

RF Propagation

  • Wall attenuation: 3-8 dB (drywall), 10-15 dB (concrete)
  • Multipath effects in hallways
  • Furniture/equipment obstruction
  • Dynamic environment (people, doors)

Cost Optimization

  • Minimize total anchor count
  • Prefer PoE over AC adapters (reduce labor)
  • Cluster installations for efficiency
  • Balance coverage vs budget constraints
